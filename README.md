# HRMS Mobile App - Login Screen

This project is a React Native Expo application featuring a minimalist and modern Login UI.

## Features

- **Minimalist Design**: Clean interface with modern typography and whitespace.
- **Interactive Inputs**: Focused states with shadow effects and icon color changes.
- **Secure Password**: Toggle visibility for password input.
- **Responsive Layout**: Works on both iOS and Android with `SafeAreaView` and `KeyboardAvoidingView`.
- **Custom Button**: Gradient button for a modern look.

## Prerequisites

- Node.js installed.
- Expo Go app on your mobile device (or an emulator).

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd hrms-app
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npx expo start
    ```

4.  Scan the QR code with the Expo Go app (Android) or Camera app (iOS).

## Project Structure

- `src/screens/LoginScreen.js`: The main login screen component.
- `App.js`: The entry point of the application.
